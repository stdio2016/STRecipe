# STRecipeMOD

English version see [here](https://github.com/stdio2016/shapeclear-html/blob/master/README_en.md)

這是我和hokomili做的，*超*簡單的Minecraft MOD，支援Minecraft 1.11.2

## 功能

這個MOD只增加合成公式和熔煉公式，不過有的公式實在太**OP**了，會*破壞*遊戲平衡，安裝時請警慎小心。

記得裝Forge
